# Axiom

> ⚠️ This project is currently under active development and not yet ready for production use.

Axiom is an open source .NET application framework aimed at providing a clean, modular foundation for building modern applications. It covers common cross-cutting concerns such as dependency injection, modularity, hosting lifecycle, and AOP-style interception — letting you focus on your application logic instead of infrastructure boilerplate.

For more information, visit [axiomframework.dev](https://axiomframework.dev).
