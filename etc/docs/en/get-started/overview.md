# Overview

Axiom is an open source .NET application framework for building modular applications. It provides foundational infrastructure for dependency injection, modularity, hosting lifecycle, and AOP-style interception, letting you focus on your application logic instead of infrastructure boilerplate.

::: danger ❗Under development
Axiom is in early alpha. APIs may change without notice and is not recommended for production use.
:::
